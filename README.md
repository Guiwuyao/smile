# 🌟 专属快乐魔法棒 (Happy Magic Wand)

这是一个为了哄最好朋友开心而专属定制的“快乐盲盒”网页。当朋友心情低落、头顶有乌云时，只需点击按钮，就能收获专属的治愈文案、搞笑表情包、欢快的背景音乐以及满屏的彩纸特效！🎉

希望这个小小的网页，能成为她随时随地可以访问的秘密避风港。

## ✨ 核心功能

* **🎈 惊喜彩纸：** 每次点击按钮都会触发满屏的 Confetti 庆祝特效。
* **💌 情绪盲盒：** 内置多条走心与搞笑并存的专属文案，每次点击随机抽取。
* **🖼️ 快乐源泉：** 自动从图库中随机抽取动图或表情包同步弹出。
* **🎵 专属 BGM：** 伴随魔法开启，后台会自动播放她最喜欢的背景音乐。

## 📁 项目文件结构

* `index.html` - 网页的主体骨架。
* `style.css` - 网页的颜值和排版（包括图片的大小控制）。
* `script.js` - 网页的互动逻辑（控制特效、音乐、随机抽取文案和图片）。
* `bgm.mp3` - 隐藏的背景音乐文件。
* `gifs/` - 存放所有表情包和动图的专属文件夹。
* `generate_memes.py` - 用于自动化处理图片的 Python 脚本。

## 🛠️ 如何更新表情包图库？

为了方便日后添加新的搞笑图片，本项目配备了自动化脚本。更新图片的步骤如下：

1. 将新的表情包或动图放入 `gifs/` 文件夹中。
2. 在本地电脑的项目根目录下，打开终端/命令行。
